What we are changing and why

Integration pages today are partner listing templates. They convert poorly on organic because they do not answer the actual query a dev has: how do I add web search to my OpenAI agent in 30 seconds? The deepening rewrite makes that the page.

  • Title and H1 lead with the head term, not the brand pair. “OpenAI Web Search Tool with Exa” vs “Exa + OpenAI”.
  • Code snippet front and center. Above the fold, 3 line setup.
  • How to walkthrough with HowTo schema. Step 1, 2, 3 earns rich snippet eligibility and Gemini and ChatGPT citation.
  • Six question FAQ with FAQPage schema. Same playbook as /exa-api.

Quickstart in 3 lines

The Exa SDK ships the OpenAI function spec for you. Dev visits cold, sees the snippet, ships in 5 minutes.

from openai import OpenAI from exa_py import Exa openai = OpenAI() exa = Exa(api_key="...") # Register Exa as an OpenAI tool tools = [exa.openai_tool()] response = openai.chat.completions.create( model="gpt-4o", messages=[{"role": "user", "content": "What launched today in AI?"}], tools=tools, )

How to add OpenAI web search with Exa

HowTo schema attached. Three steps eligible for rich result.

Step 1. Install the SDKs. pip install openai exa_py

Step 2. Register the Exa tool with your OpenAI client. exa.openai_tool()

Step 3. Call chat.completions.create with tools=tools and let GPT-4o trigger the search when it needs the web.

New section

Frequently asked questions

Does OpenAI have a built in web search API?

OpenAI ships browsing inside ChatGPT but the API does not have a first party web search endpoint. Use a search tool like Exa.

How do I add web search to GPT-4o?

Call exa.openai_tool() and pass the result as a tool in your OpenAI chat completion. GPT-4o decides when to call it.

What does the OpenAI plus Exa integration cost?

OpenAI tokens at OpenAI rates plus Exa per request pricing. Most agents land under $0.02 per turn.

Can I use Exa with the OpenAI Assistants API?

Yes. Register Exa as a function tool on the assistant.

How does Exa's OpenAI integration compare to Tavily?

Same call signature pattern. Exa wins on latency and full page content. See /versus/tavily.

Does Exa support streaming responses with OpenAI?

Yes. Streaming flows through OpenAI's stream and the Exa tool call resolves inline.
