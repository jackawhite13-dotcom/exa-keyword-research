What we are changing and why

The live page already names the right primitives. ExaSearchRetriever for RAG pipelines, ExaSearchResults for agents, LangGraph compatibility. What it does not do is rank. The page is a partner listing template with no code on page and no schema.

  • Title and H1 lead with the head term. “LangChain Web Search Tool with Exa” vs “Exa + LangChain”.
  • Code snippet front and center. ExaSearchRetriever in 4 lines.
  • HowTo and FAQPage schema on the new walkthrough and FAQ.
