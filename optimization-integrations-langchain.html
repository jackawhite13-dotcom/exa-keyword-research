New section

Frequently asked questions

How do I add web search to a LangChain agent?

Initialize ExaSearchResults, register as a tool on your agent, then call invoke().

What is the difference between ExaSearchRetriever and ExaSearchResults?

Retriever fits RAG pipelines and returns Documents. Results is for agent tool use and returns structured results with metadata.

Does Exa work with LangGraph?

Yes. Drop ExaSearchResults into any LangGraph node.

How does Exa compare to LangChain's built in search tools?

LangChain ships generic search wrappers. Exa adds neural retrieval, 1,200 domain filters, and full page content with highlights.

Can I use Exa with streaming and async LangChain chains?

Yes. Async support and streaming both included.

What does the LangChain plus Exa integration cost?

Exa per request pricing. No LangChain license required.
