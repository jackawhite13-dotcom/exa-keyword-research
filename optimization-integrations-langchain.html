NewCodeAbove-the-fold setup snippet
from langchain_exa import ExaSearchRetriever
from langchain_openai import ChatOpenAI

retriever = ExaSearchRetriever(exa_api_key="...", k=5)
llm = ChatOpenAI(model="gpt-4o")

docs = retriever.invoke("What launched today in AI?")
answer = llm.invoke(
    f"Using these sources, answer the question:\n{docs}"
)

Four lines of setup, real call. Optional second snippet for the LangGraph tool wiring (ExaSearchResults).
