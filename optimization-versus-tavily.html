01

What we are changing and why

The page is accuracy led with strong benchmark tables (WebWalker 81% vs 71%, MKQA 70% vs 63%) and the right key differentiator H3s. Three things are missing.

  • Post Nebius positioning. Tavily was acquired by Nebius. Buyers researching alternatives need to see what that means for their roadmap exposure.
  • Migration path. The page ends in a generic “Try API for free” CTA. A migration code diff is the most useful conversion element for active evaluation buyers.
  • FAQPage schema. The page has a FAQ but no JSON-LD. Single edit, unlocks AI overview citation.
