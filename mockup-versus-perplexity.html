Sonar Pro, what it is and when to use it

Sonar is Perplexity's LLM synthesized answer endpoint. Sonar Pro adds longer outputs and stronger reasoning at a higher price tier. Both are good for chat style answers but trade transparency for synthesis. You get a generated response instead of the retrieval results you can re rank, filter, or feed into your own LLM. Exa's API returns the underlying retrieval so your agent owns the synthesis step.

What is the difference between Perplexity Sonar and Sonar Pro?

Sonar is the base LLM synthesized answer endpoint. Sonar Pro adds longer outputs and stronger reasoning at a higher price tier.

How does Exa compare to Perplexity Sonar?

Exa returns retrieval results that your agent owns. Sonar returns a synthesized answer.

How does Exa's pricing compare to Perplexity API?

Flat per request pricing on Exa. Perplexity bills across two pricing surfaces (Search API + Sonar models).

Can I switch from Sonar to Exa without rewriting my prompt logic?

Yes. Use Exa to retrieve, then your existing LLM call to synthesize. Migration takes one afternoon.
